# 🖤 The Oracle Adventure 🖤

A **mini interactive Python story** where your choices shape your fate.  
Answer questions, make decisions, and discover your destiny in a mystical world.  

> “Every choice you make guides your path… choose wisely.” 🔮

---

## Features

- Interactive **terminal-based Python story**  
- **5+ scenes** with choices affecting the outcome  
- Multiple endings based on **your decisions and score**  
- Immersive storytelling with **emojis and descriptive text**  
- Beginner-friendly code, easy to read and expand
